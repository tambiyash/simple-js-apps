# VanillaJS

A personal collection of small vanilla JavaScript projects and UI demos. Each folder contains a standalone static example built with plain HTML, CSS, and JavaScript.

## Projects

- `array-cardio/day-1` — array practice exercises.
- `css-variables` — CSS custom properties demo.
- `drum-kit` — interactive keyboard/mouse drum kit.
- `flex-panel-gallery` — responsive flex panel gallery.
- `geo-growth-population` — population and growth visualization example.
- `global-search-js` — searchable list / global search demo.
- `html-canvas` — HTML canvas drawing and animation.
- `js-clock` — JavaScript clock demo.
- `pagination-fetch-button` — fetch data with pagination via button.
- `pagination-fetch-scroll` — fetch data with infinite scroll.
- `push-notifications-local` — local browser push notification demo.

## Usage

Open any project folder and launch its `index.html` file in a browser. For best results, use a local web server when examples rely on network requests or browser APIs.

### Run a simple local server

From the repository root:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` and navigate to the project folder.

## Notes

- No build step is required.
- All examples use plain JavaScript and web platform APIs.
